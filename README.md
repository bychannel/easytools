## 💡  简介

[easytools](https://pkg.go.dev/github.com/bychannel/easytools) 是一款 Go 语言编程常用工具库，主要使用在 Go 游戏开发方面，提供了简洁易用、同时高效安全的操作方法，避免编写无效的重复代码，提高编程效率，让 Go 编程更加的丝滑。

## ✨ 功能

* stringtools 字符串工具类
* timetools 时间工具类
